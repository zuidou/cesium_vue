# Cesium_Vue 框架整合（vue3.x+vue/cli5.0+cesium 1.86.1）


## 一、框架依赖

 "cesium": "^1.86.1",

  "core-js": "^3.8.3",

  "vue": "^3.2.6",

  "vue-router": "^4.0.3",

  "vuex": "^4.0.0"

"@vue/cli-plugin-vuex": "~5.0.0-beta.6",

  "@vue/cli-service": "~5.0.0-beta.6",

## 二、项目操作

项目依赖包安装

```
npm install
```

项目编译

```
npm run serve
```

项目打包

```
npm run build
```
